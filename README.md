# Downloads Folder Cleanup

This Python script automates the cleanup of your Downloads folder by organizing files into different categories by avoiding downloads folder clutter.

## Features

- Organizes files into categories (Documents, Images, Others).

## Getting Started

### Prerequisites

- Python 3.x installed on your system.

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/nimscodes/downloads_folder_cleanup.git
   ```

2. Navigate to the project directory

   ``` bash
   cd downloads_folder_cleanup
   ```

3. Create a virtual environment (optional but recommended)
4. 
5. ```bash
   python -m venv name_of_virtual_environment
   source name_of_virtual_ennvironment/bin/activate # On linux or macOS
   name_of_virtual_environment\Scripts\activate # On windows
   ```


# Usage

1. Run the script

```bash
python cleanup_downloads.py
```
